# RentTech – Сервис аренды спецтехники

**RentTech** – это веб-приложение для аренды строительной и специальной техники. Платформа позволяет пользователям находить технику, бронировать её, управлять своими арендами, ставить оценки, а администраторам – управлять клиентами, техникой и бронированиями. Встроенный модуль прогнозирования спроса на основе модели Хольта-Уинтерса помогает планировать загрузку парка и оптимизировать ценообразование.

---

## 🚀 Основные возможности

### Для всех пользователей
- **Каталог техники** – фильтрация по типу, цене, году, статусу, поиск по названию.
- **Детальный просмотр** техники с фотографией типа, характеристиками и формой бронирования.
- **Авторизация и регистрация** (JWT-токены, пароли хешируются BCrypt).
- **Аналитика** – популярная техника, выручка за период, просроченное ТО.

### Для авторизованных пользователей
- **Личный кабинет** – история бронирований, отмена активных, продление на 1 час, оплата аренды.
- **Оценка техники** – можно ставить оценки только технике, которую уже арендовали (аренда завершена).
- **Моя техника** – добавление собственной техники в каталог, редактирование и удаление (только для владельца).

### Для администраторов
- **CRUD клиентов** – просмотр, добавление, редактирование, удаление.
- **CRUD техники** – управление всеми единицами техники (админ может назначать владельца).
- **CRUD бронирований** – просмотр, создание, редактирование, удаление любых бронирований.
- **Модальные окна** для удобного редактирования (без перезагрузки страницы).

### Модуль прогнозирования спроса
- **Модель Хольта-Уинтерса** (аддитивная сезонность, период 7 дней) – прогноз количества аренд на каждый день.
- **Простой прогноз** (по средним дням недели) – для сравнения.
- **Настройка параметров** модели (α, β, γ) прямо в интерфейсе аналитики.
- **Визуализация** – график и таблица прогнозных значений.
- Прогноз строится на основе **завершённых** аренд (статус `"завершено"`).

## 🛠 Технологии

### Бэкенд
- **ASP.NET Core 9.0** – Web API
- **Entity Framework Core 9.0** – ORM
- **Npgsql** – провайдер для PostgreSQL
- **JWT** – аутентификация
- **BCrypt.Net-Next** – хеширование паролей
- **Swagger/OpenAPI** – документация API (в режиме разработки)

### Фронтенд
- **HTML5 + CSS3** (адаптивная вёрстка)
- **Bootstrap 5** – стилизация и компоненты
- **Chart.js** – построение графиков прогноза
- **Vanilla JavaScript** (весь клиентский код в одном `app.js`)
